## Running the project

- Clone the web project:
  ```bash
  git clone https://github.com/emanueltavecia/foodexplorer-web.git
  ```
- Access project folder:
  ```bash
  cd foodexplorer-web
  ```
- Install dependencies:
  ```bash
  npm install
  ```
- Run project:
  ```bash
  npm run dev
  ```
- ðŸŽ‰ Finished, now you can access "http://localhost:5173/" on your browser and test the application.
